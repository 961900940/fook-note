###什么是回调函数

1. 用户将自定义函数赋值给变量。
2. 对象中的静态方法也是回调函数。

关键字：is_callback、callback

```php
class classname{
    public function name(callback $callback){	//限定参数类型必须为回调函数
	//判断$callback是否为回调函数。请忽略逻辑重复，主要讲的是这个意思。
	if(is_callback($callback){}
    }
}
```
